Ai - arcade game

Ai is a simple, classic, arcade game for everyone. The rules of the game is simple: you need to clear picture by controlling a fairy. Sounds easy, but different obstacles in a levels makes this game quite challeging and fun.

Features: ★ Easy to start playing ★ Over a 100 handmade levels ★ Select level and play ★ Two different ways to play ★ Challenges! Endless game mode!
